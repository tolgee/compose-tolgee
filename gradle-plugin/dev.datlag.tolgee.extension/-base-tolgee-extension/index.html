BaseTolgeeExtension

open class BaseTolgeeExtension(objectFactory: ObjectFactory)(source)

Base class for configuring global options for the Tolgee Gradle plugin.

This class provides common configuration properties that can be shared across Gradle tasks or extensions inheriting from it. It allows setting up essential options for interacting with the Tolgee CLI or REST API.
