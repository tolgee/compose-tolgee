State

sealed class State : CharSequence(source)

Represents the state of a translation in Tolgee.

This sealed class is used to specify the required state of translations when pulling strings from Tolgee. It supports predefined states (e.g., UNTRANSLATED, TRANSLATED, etc.) and allows for custom states through the Custom data class.
